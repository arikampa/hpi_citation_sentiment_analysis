Files produced:
- long_shap_token_categories.csv: Long-format token-category dataset in high-pressure region.
- per_model_C3_vs_C4_chi_square_results.csv: For each model, chi-square test comparing token-category distribution between C3 and C4.
- per_case_models_vs_categories_chi_square_results.csv: For each case (C3/C4), chi-square test comparing token-category distribution across models.
- focused_C3_vs_C4_token_category_tests_per_model.csv: 2x2 tests per model and token category (C3 vs C4).
- descriptive_category_proportions_by_model_case.csv: Descriptive proportions for reporting.
